# 产品说明文档 (产品经理版)

## 1. 产品概述
**思辨场** 是一款多模型并行评测与内容生产工具。它允许用户输入单一提示词（Prompt），同时调用多个主流大语言模型（LLM）进行回答，并提供自动化对比与回答整合功能。

## 2. 核心功能及形态

### 2.1 提示词输入
*   **输入形式**：支持多行文本输入的编辑框。
*   **操作交互**：支持回车换行，Command/Ctrl + 回车快速发送。支持示例填充与一键清空。

### 2.2 模型管理与选择
*   **模型注册**：集成包括 OpenAI, Anthropic, Gemini, xAI 等国际模型，以及月之暗面 (Kimi), 智谱清言 (GLM), MiniMax, 通义千问, DeepSeek 等国产模型。
*   **筛选机制**：提供“中国”、“美国”、“全部”三个地域维度的快速过滤标签。
*   **多选操作**：支持模型项的独立勾选或全局全选/全不选。

### 2.3 响应呈现
*   **并行输出**：多个模型回答以卡片形式横向排列，支持流式渲染（Streaming）。
*   **内容操作**：
    *   **排序调整**：卡片支持左/右平移调整显示位置。
    *   **一键复制**：每个完成的响应卡片右上角设有“复制”键，点击即可提取全文。
*   **运行反馈**：实时显示每个模型的请求状态（请求中、完成、失败）及响应耗时（毫秒级）。

### 2.4 智能汇总 (Post-processing)
*   **差异总结**：调用 DeepSeek 模型对所有生成答案进行多维度（风格、结构、信息密度等）对比总结。
*   **最佳答案融合**：调用 Qwen 模型整合多个答案的优点，输出一个结构清晰、无冗余的终版答案。
*   **总结交互**：支持重新生成及一键复制总结内容。

### 2.5 记录与存储
*   **历史搜索**：支持根据关键词搜索历史对话记录，支持单条删除或全局清空。
*   **数据持久化**：
    *   本地缓存：使用浏览器存储（LocalStorage）记录最近的历史记录。
    *   文件同步：支持通过现代浏览器 File System Access API 绑定本地 `.jsonl` 文件，实现准实时的数据物理备份。

### 2.6 辅助工具
*   **模型能力榜**：接入第三方独立测评数据（Artificial Analysis）的柱状图看板，分类展示中美模型的智力指标排名。
*   **个性化设置**：支持用户自定义上传背景图片。

## 3. 业务目标
*   **效率提升**：减少在不同模型界面间切换、复制、粘贴的重复劳动。
*   **择优参考**：通过直观对比，帮助用户筛选出最符合需求的模型回答。
*   **深度洞察**：通过自动总结，发现不同模型在处理特定任务时的能力偏好。
